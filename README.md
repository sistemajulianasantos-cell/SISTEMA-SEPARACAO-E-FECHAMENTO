# Sistema de Separação e Fechamento
Romero Coquetéis & Eventos

## 📋 Como começar

### 1. Baixe os arquivos
- `index.html` - arquivo principal do projeto
- `README.md` - este arquivo
- `.gitignore` - configuração do Git (opcional)

### 2. Organize a pasta
```
sistema-separacao-e-fechamento/
├── index.html
├── README.md
└── .gitignore
```

### 3. Teste no navegador
- Abra o arquivo `index.html` no seu navegador
- Você deve ver a mensagem: **✅ Firebase conectado com sucesso!**
- Abra o Console (F12) para confirmar

### 4. Conecte ao GitHub

Se já tem Git instalado, abra o Prompt de Comando na pasta do projeto:

```bash
git init
git add .
git commit -m "Projeto inicial com Firebase"
git branch -M main
git remote add origin https://github.com/seuusuario/sistema-separacao-e-fechamento.git
git push -u origin main
```

### 5. Ative GitHub Pages
1. Vá para **Settings** do repositório no GitHub
2. Menu lateral: **Pages**
3. **Source**: escolha `main` branch
4. **Folder**: escolha `/ (root)`
5. Salve

Seu projeto estará em: `https://seuusuario.github.io/sistema-separacao-e-fechamento/`
